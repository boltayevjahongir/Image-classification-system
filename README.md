
# Image Classification System

## Requirements
- Python 3.8+
- PyTorch
- Torchvision
- Pillow

## Installation
Run:
```
pip install -r requirements.txt
```

## Usage
To classify an image:
```
python classify_image.py /path/to/image.jpg
```

### Supported:
- Formats: `.jpg, .jpeg, .png`
- Max Size: 10 MB
